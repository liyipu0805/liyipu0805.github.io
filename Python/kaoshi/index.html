四. HTML

4.1 HTTP协议的工作方式是什么？它主要包括哪几个过程？

HTTP 采用请求-响应的通讯方式，首先在客户端和服务器之间建立连接，然后，
客户端发送请求到服务器，接着，服务器发送响应到客户端，最后通讯结束并关闭连接。

4.2 HTTP请求报文的格式是什么？在请求方式中，GET方法和POST方法的区别是什么？

分为请求行、请求头部、请求数据。
GET方法：获取一个指定URL的全部内容。
POST方法：想服务器端发送附加数据。
区别在于用户是否传送数据。

4.3 HTTP响应报文额的格式是什么？在请求方法中，GET方法和POST方法的区别是什么？

分为状态行，响应头部，响应正文
200：客户端请求成功。
400：客户端请求有语法错误。
401：请求未经授权。
403：服务器拒绝提供服务。
404：请求资源不存在。
500：服务器发生不可预期的错误。

4.4 一个HTML文档主要包括哪几个部分？什么是HTML元素？

主要包括html、head和body。html的元素指的是从开始标签到结束标签的所有代码。

4.9 什么是多线程，与多进程的区别是什么？

多线程是指从软件或者硬件上实现多个线程并发执行的技术。
1.一个线程只能属于一个进程，而一个进程可以有多个线程，但至少有一个线程。
2.资源分配给进程，同一进程的所有线程共享该进程的所有资源。
3.CPU分给线程，即真正在CPU上运行的是线程。

反爬虫机制：使用任何技术手段，阻止别人批量获取自己网站信息的一种方式，主要组织自动爬虫程序访问。 分为三类：

  1. 基于身份识别的反爬。例如：通过Headers、请求参数反爬。
  2. 基于爬虫行为进行反爬。例如：基于请求频率或请求总数量。
  3. 基于数据加密进行反爬。例如：对响应中含有的数据进行特殊化处理。

五、数据挖掘基础

5.1 请简述在数据挖掘中什么是训练集、验证集和测试集。

训练集指参与到模型训练的数据。
验证集指模型训练过程中用于评估模型性能的数据，用于指导超参数的调节。
测试集指模型训练完成后用于测试模型的泛化误差。

5.2 评估机器学习模型的时候，什么是准确率？什么是召回率？

准确率（Precision）：预测为正的样本中有多少是真正的正样本。 TP / (TP + FP)
召回率（Recall）: 样本中的正例有多少被预测正确。 TP / (TP + FN)

5.3 常见的数据挖掘流程是什么？

  1. 数据获取
  2. 数据清洗
  3. 特征工程
  4. 模型构建
  5. 模型训练
  6. 模型评估

5.4 交叉验证的作用是什么？K折交叉验证的步骤是什么？

交叉验证可以更准确的对模型评估，防止由于验证集数据划分导致评价指标的特例。

  1. 将全部训练集 S分成 k个不相交的子集，假设 S中的训练样例个数为 m，那么每一个子 集有 m/k 个训练样例，，相应的子集称作 {s1,s2,…,sk}。
  2. 每次从分好的子集中里面，拿出一个作为测试集，其它k-1个作为训练集
  3. 根据训练训练出模型或者假设函数。
  4. 把这个模型放到测试集上，得到分类率。
  5. 计算k次求得的分类率的平均值，作为该模型或者假设函数的真实分类率。
